# D60 Restaurant, Website Mockup

Free pitch mockup for **D60 Restaurant**, a husband and wife, Iranian family owned
spot on Barton Street in Hamilton. Sent to the owner to gauge interest in building
a full site with Anchor and Pixel.

## Files

```
D60/
  index.html      The full single-page site (self contained CSS and JS)
  images/         20 real food photos pulled from the current d60.ca menu
  server.js       Tiny local preview server (Node)
  README.md       This file
```

## How to preview

- Quickest: double click `index.html`.
- With a local server: run `node server.js` in this folder, then open
  `http://localhost:4322`.

## Design notes

- Direction: dark cinematic. Near black background, saffron gold and pomegranate
  accents, ivory text. Fonts are Fraunces (display) and Manrope (body).
- Built against the eight point "$10K Checklist": committed art direction, real
  typography, restrained color, breathing hierarchy, real photography, quiet motion,
  a mobile layout that is designed rather than shrunk, and the invisible basics
  (semantic HTML, WCAG AA contrast, keyboard support, fast load).
